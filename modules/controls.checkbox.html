Checkbox:New (checked, caption, textalign, boxalign, boxsize, textColor, OnChange)
Checkbox fields. Inherits from Control.

Parameters:

  • checked bool checkbox checked state, default: true
  • caption string caption to appear in the checkbox, default: "text"
  • textalign string text alignment, default: "left"
  • boxalign string box alignment, default: "right"
  • boxsize int box size, default: 10
  • textColor text color table {r,g,b,a}, default: {0,0,0,1}
  • OnChange listener functions for checked state changes
